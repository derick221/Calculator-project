# Expense and Calories Tracker

Tracker is a Python project developed as part of a task assigned by Yandex.Praktikum. It is designed to help users track their expenses and calories intake.

The project utilizes object-oriented programming (OOP) principles.

## Features

- Record and store expenses with details like amount, comment, and date.
- Calculate and display daily and weekly statistics.
- Manage expenses in different currencies.
- Track calorie intake and set limits for daily calorie consumption.
